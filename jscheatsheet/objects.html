Objects Cheat Sheet

Objects in JavaScript are collections of key-value pairs that allow you to store and organize related data in a structured way. Unlike arrays, which use numeric indices, objects use named properties (keys) to access their values. You create objects using curly braces, and each property consists of a key and its associated value, separated by a colon. Objects are ideal for representing real-world entities with multiple attributesâ€”for example, a user object might have properties like name, email, and age. They're fundamental to JavaScript programming and form the basis of more complex concepts like classes and prototypes, making them essential for organizing data and building applications.

Creation & Interfaces

// JavaScript
const candy = { name: 'Gummy Bear', flavor: 'Cherry' };

// TypeScript
interface Candy {
  name: string;
  flavor: string;
  rating?: number; // Optional property
}

const treat: Candy = { name: 'Lollipop', flavor: 'Strawberry' };

Accessing & Manipulating

Accessing properties

console.log(candy.name); // Dot notation
console.log(candy['flavor']); // Bracket notation

Adding/Updating

candy.color = 'Red';
candy.flavor = 'Sour Cherry';

Deleting

delete candy.color;

Object Methods

Object.keys(obj): Get all property names

const keys = Object.keys(candy); // ['name', 'flavor']

Object.values(obj): Get all property values

const values = Object.values(candy); // ['Gummy Bear', 'Cherry']

Object.entries(obj): Get key-value pairs

for (const [key, value] of Object.entries(candy)) {
  console.log(`${key}: ${value}`);
}

Object.assign(target, ...sources): Merge objects

const fullTreat = Object.assign({}, candy, { size: 'Large' });

Spread Operator

Shallow copy & merging

const copiedCandy = { ...candy };
const mergedCandy = { ...candy, brand: 'SweetCo' };
