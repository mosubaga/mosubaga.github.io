Sweet Typescript & Javascript Cheat Sheets

Welcome to the most delicious resource for mastering TypeScript and JavaScript! Explore our sugary cheat sheets below.

JavaScript, created by Brendan Eich in just 10 days back in 1995, has evolved from a simple scripting language for adding basic interactivity to web pages into one of the most influential programming languages in the world. Originally designed to run exclusively in web browsers, JavaScript now powers everything from server-side applications (thanks to Node.js) to mobile apps, desktop software, and even IoT devices. As JavaScript projects grew more complex, Microsoft introduced TypeScript in 2012 as a superset of JavaScript that adds optional static typing and modern development tools, helping developers catch errors before runtime and build more maintainable large-scale applications. Today, learning JavaScript and TypeScript opens doors to virtually every area of software development—it's the only language that runs natively in all web browsers, boasts the largest package ecosystem (npm), and is consistently ranked among the most popular and in-demand programming languages worldwide. Whether you're building interactive websites, full-stack web applications, mobile apps with React Native, or cloud functions, JavaScript and TypeScript provide the versatile foundation you need to bring your ideas to life.

Basic Syntax


Control Flow (If...Else)


const temperature = 25;

if (temperature > 30) {
  console.log("It's a scorcher! ☀️");
} else if (temperature > 20) {
  console.log("Perfect weather for coding. 🍦");
} else {
  console.log("Grab a sweater! 🧥");
}

Loops


// Standard for loop
for (let i = 0; i < 3; i++) {
  console.log(`Scoop number ${i + 1}`);
}

// For...of (Iterating over arrays)
const toppings = ['Sprinkles', 'Chocolate', 'Cherry'];
for (const topping of toppings) {
  console.log(`Adding ${topping}...`);
}

TypeScript Interfaces

Interfaces define the "shape" of an object, ensuring type safety.


interface Sundae {
  flavor: string;
  scoops: number;
  hasSyrup: boolean;
  extraInstructions?: string; // Optional property
}

const myOrder: Sundae = {
  flavor: 'Vanilla',
  scoops: 2,
  hasSyrup: true
};

Variables & Types


// const: can't reassign (use by default)
const name = 'Alice';

// let: can reassign (use when value changes)
let score = 0;
score = 10;

// var: old style, function-scoped (avoid in modern code)

// Common types
const message = 'hello';      // string
const count = 42;              // number
const isActive = true;         // boolean
const nothing = null;          // null
const missing = undefined;     // undefined
const items = [1, 2, 3];      // array
const user = { name: 'Bob' }; // object

Functions


// Function declaration (hoisted)
function greet(name) {
  return `Hello, ${name}!`;
}

// Arrow function (concise, no own 'this')
const add = (a, b) => a + b;

// Arrow with body
const createUser = (name, age) => {
  return { name, age };
};

// Default parameters
function serve(flavor = 'Vanilla', scoops = 1) {
  return `${scoops} scoop(s) of ${flavor}`;
}
