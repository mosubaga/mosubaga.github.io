As JavaScript projects grew more complex, Microsoft introduced TypeScript in 2012 as a superset of JavaScript that adds optional static typing and modern development tools, helping developers catch errors before runtime and build more maintainable large-scale applications.   console.log(`Adding ${topping}...`);
}

TypeScript Interfaces

Interfaces define the "shape" of an object, ensuring type safety.


interface Sundae {
  flavor: string;
  scoops: number;
  hasSyrup: boolean;
  extraInstructions?: string; // Optional property
}

const myOrder: Sundae = {
  flavor: 'Vanilla',
  scoops: 2,
  hasSyrup: true
