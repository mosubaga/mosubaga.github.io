Arrays Cheat Sheet

Arrays are one of the most fundamental and versatile data structures in JavaScript and TypeScript, serving as ordered collections that can store multiple values in a single variable. Unlike arrays in many other programming languages, JavaScript arrays are remarkably flexible—they can hold elements of different types simultaneously, grow and shrink dynamically without pre-declaring a size, and even contain other arrays or complex objects. TypeScript enhances this foundation by allowing you to specify the types of elements an array should contain, providing helpful autocomplete suggestions and catching type-related errors during development. Whether you're managing a list of user names, storing numerical data for calculations, organizing API responses, or building complex data structures, arrays provide powerful built-in methods like map(), filter(), reduce(), and forEach() that make data manipulation elegant and efficient. Mastering arrays is essential for any JavaScript or TypeScript developer, as they appear in virtually every application—from simple todo lists to sophisticated data processing pipelines—and form the building blocks for more advanced programming patterns and algorithms.

Creation & Basics

// JavaScript
const fruits = ['Apple', 'Banana'];

// TypeScript
const berries: string[] = ['Strawberry', 'Blueberry'];
const numbers: Array<number> = [1, 2, 3];

Adding/Removing Elements

push(...items): Add to end

fruits.push('Orange'); // ['Apple', 'Banana', 'Orange']

pop(): Remove from end

const last = fruits.pop(); // 'Orange'

unshift(...items): Add to beginning

fruits.unshift('Mango'); // ['Mango', 'Apple', 'Banana']

shift(): Remove from beginning

const first = fruits.shift(); // 'Mango'

Transformation

map(callback): Create new array by transforming each element

const lengths = fruits.map(f => f.length);

filter(callback): Create new array with elements that pass a test

const longNames = fruits.filter(f => f.length > 5);

reduce(callback, initialValue): Reduce array to a single value

const totalLength = fruits.reduce((sum, f) => sum + f.length, 0);

Searching

find(callback): Returns the first element that matches

const found = fruits.find(f => f.startsWith('A'));

includes(item): Returns true if item is in array

const hasApple = fruits.includes('Apple');
