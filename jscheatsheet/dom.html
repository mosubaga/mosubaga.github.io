DOM & Events Cheat Sheet

The DOM (Document Object Model) is the browser's representation of your HTML page as a tree of objects. Every element, attribute, and piece of text becomes a node that JavaScript can read and modify. When you change the DOM, the browser immediately updates what the user sees on screen. Understanding DOM manipulation and events is the foundation of interactive web programming — it is how buttons respond to clicks, forms validate input, and content updates without a full page reload.

Selecting Elements

querySelector(selector): Returns the first element matching a CSS selector

const btn = document.querySelector('.submit-btn');
const header = document.querySelector('#main-header');
const firstItem = document.querySelector('ul > li:first-child');

querySelectorAll(selector): Returns a NodeList of all matching elements

const items = document.querySelectorAll('.menu-item');

// Loop through all matches
items.forEach(item => {
  item.style.color = 'darkblue';
});

getElementById(id): Returns the element with the given id

const app = document.getElementById('app');
// Equivalent to querySelector('#app'), but slightly faster

getElementsByClassName(name): Returns a live HTMLCollection

const cards = document.getElementsByClassName('card');
// This is a live collection — it updates automatically
// when elements are added or removed from the DOM

Modifying Elements

textContent: Get or set the text of an element (safe, no HTML parsing)

const title = document.querySelector('h1');
title.textContent = 'Welcome!'; // Sets plain text

innerHTML: Get or set the HTML content (be careful with user input)

const list = document.querySelector('.list');
list.innerHTML = '<li>Item 1</li><li>Item 2</li>';
// Warning: never use innerHTML with untrusted user input

style: Set inline CSS styles directly

const box = document.querySelector('.box');
box.style.backgroundColor = '#fce4ec';
box.style.padding = '1rem';
box.style.borderRadius = '8px';

classList: Add, remove, toggle, or check CSS classes

const nav = document.querySelector('.navbar');

nav.classList.add('sticky');        // Add a class
nav.classList.remove('hidden');     // Remove a class
nav.classList.toggle('open');       // Toggle on/off
nav.classList.contains('sticky');   // true

setAttribute / getAttribute: Work with HTML attributes

const link = document.querySelector('a');
link.setAttribute('href', 'https://example.com');
link.setAttribute('target', '_blank');

const url = link.getAttribute('href'); // 'https://example.com'

Creating & Removing Elements

createElement / appendChild: Create and add a new element

const ul = document.querySelector('.todo-list');
const li = document.createElement('li');
li.textContent = 'Buy milk';
li.classList.add('todo-item');
ul.appendChild(li); // Adds to the end of the list

prepend: Insert as the first child

const container = document.querySelector('.messages');
const notice = document.createElement('div');
notice.textContent = 'New message!';
container.prepend(notice); // Adds to the beginning

remove(): Remove an element from the DOM

const popup = document.querySelector('.popup');
popup.remove(); // Gone from the page

replaceWith(newElement): Replace an element with another

const oldBtn = document.querySelector('.old-button');
const newBtn = document.createElement('button');
newBtn.textContent = 'Click Me';
oldBtn.replaceWith(newBtn);

cloneNode(deep): Clone an element (pass true to include children)

const template = document.querySelector('.card-template');
const copy = template.cloneNode(true); // Deep clone
copy.querySelector('h2').textContent = 'New Card';
document.querySelector('.grid').appendChild(copy);

Event Handling

addEventListener(event, handler): Attach an event listener

const btn = document.querySelector('.btn');
btn.addEventListener('click', () => {
  alert('Button clicked!');
});

Event object: Access details about the event

document.addEventListener('click', (event) => {
  console.log(event.target);    // The element that was clicked
  console.log(event.type);      // 'click'
  console.log(event.clientX);   // Mouse X position
});

Common events: Frequently used event types

// Page loaded and DOM ready
document.addEventListener('DOMContentLoaded', () => {
  console.log('DOM is ready');
});

// Keyboard events
document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') closeModal();
});

// Input event — fires on every keystroke
const searchInput = document.querySelector('#search');

function closeModal() {
  console.log('Closing modal');
}

function filterResults(query) {
  console.log('Filtering for:', query);
}

searchInput.addEventListener('input', (e) => {
  filterResults(e.target.value);
});

Event delegation: Handle events on dynamic or many child elements

// Instead of adding listeners to each button,
// listen on the parent and check event.target
const list = document.querySelector('.todo-list');
list.addEventListener('click', (e) => {
  if (e.target.matches('.delete-btn')) {
    e.target.closest('li').remove();
  }
  if (e.target.matches('.toggle-btn')) {
    e.target.closest('li').classList.toggle('done');
  }
});

Forms & User Input

Reading input values: Get what the user typed

const nameInput = document.querySelector('#name');
const email = document.querySelector('#email');

console.log(nameInput.value);  // Current text in the input
console.log(email.value);

// For checkboxes and radio buttons
const agree = document.querySelector('#agree');
console.log(agree.checked); // true or false

Handling form submission: Prevent default and process data

const form = document.querySelector('#signup-form');
form.addEventListener('submit', (e) => {
  e.preventDefault(); // Stop the page from reloading

  const formData = new FormData(form);
  const name = formData.get('name');
  const email = formData.get('email');

  console.log({ name, email });
  // Send to server with fetch()...
});

Basic form validation: Check input before submitting

const form = document.querySelector('#signup-form');

form.addEventListener('submit', (e) => {
  e.preventDefault();

  const email = document.querySelector('#email');
  const errorMsg = document.querySelector('.error');

  if (!email.value.includes('@')) {
    errorMsg.textContent = 'Please enter a valid email';
    email.classList.add('input-error');
    return;
  }

  errorMsg.textContent = '';
  email.classList.remove('input-error');
  form.submit();
});

Traversing the DOM

parentElement: Get the parent of an element

const item = document.querySelector('.list-item');
const list = item.parentElement; // The <ul> or <ol>

children: Get all child elements

const nav = document.querySelector('nav');
const links = nav.children; // HTMLCollection of child elements
console.log(links.length);  // Number of direct children

nextElementSibling / previousElementSibling: Navigate between siblings

const current = document.querySelector('.active');
const next = current.nextElementSibling;
const prev = current.previousElementSibling;

// Move "active" class to the next sibling
if (next) {
  current.classList.remove('active');
  next.classList.add('active');
}

closest(selector): Find the nearest ancestor matching a selector

// Useful with event delegation
document.addEventListener('click', (e) => {
  const card = e.target.closest('.card');
  if (card) {
    card.classList.toggle('selected');
  }
});
