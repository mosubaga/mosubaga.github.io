Intermediate Python

Welcome to the intermediate Python guide. This site deepens core language knowledge and helps you write cleaner, faster, and more maintainable code. Each page focuses on a practical topic with concepts, key syntax, and real-world use cases.

Topics include object-oriented programming, advanced data structures, decorators and scope, generators for memory efficiency, and concurrency using both threads/processes and asyncio.

How to use this guide

Start with OOP and Collections for strong foundations, then explore Functions and Generators before moving into Concurrency and Async for performance and scalability.

Quick Map

Sample Code

# A tiny taste of intermediate Python
from dataclasses import dataclass

@dataclass
class Task:
    name: str
    done: bool = False

tasks = [Task("read docs"), Task("build project", True)]
pending = [t for t in tasks if not t.done]
print(pending)
