Linux Commands

Essential terminal commands for software engineers and developers — organized by category.

File & Directory Management

ls

List files and directories in the current directory

ls -la

List all files (including hidden) with detailed info and permissions

ls -lh

List files with human-readable file sizes (KB, MB, GB)

pwd

Print the current working directory path

cd [directory]

Change to the specified directory

cd ..

Move up one directory level

cd ~

Go to the home directory

mkdir [directory]

Create a new directory

mkdir -p [path/to/dir]

Create nested directories, including all intermediate ones

touch [file]

Create an empty file or update an existing file's timestamp

cp [source] [dest]

Copy a file to a new location

cp -r [source] [dest]

Recursively copy a directory and all its contents

mv [source] [dest]

Move or rename a file or directory

rm [file]

Remove (delete) a file

rm -r [directory]

Recursively remove a directory and its contents

rm -rf [directory]

Force-remove a directory without confirmation prompts

ln -s [target] [link]

Create a symbolic (soft) link pointing to target

find [path] -name "[pattern]"

Find files matching a name pattern under the given path

find . -type f -mtime -7

Find files modified within the last 7 days

find . -size +100M

Find files larger than 100 MB

File Viewing & Reading

cat [file]

Display the entire contents of a file

less [file]

Page through a file (q to quit, / to search)

more [file]

Display file content one screen at a time

head [file]

Show the first 10 lines of a file

head -n 20 [file]

Show the first 20 lines of a file

tail [file]

Show the last 10 lines of a file

tail -n 20 [file]

Show the last 20 lines of a file

tail -f [file]

Follow a file in real-time as new lines are added (great for logs)

wc -l [file]

Count the number of lines in a file

wc -w [file]

Count the number of words in a file

Text Processing

grep [pattern] [file]

Search for lines matching a pattern in a file

grep -r [pattern] [dir]

Recursively search for pattern in all files under a directory

grep -i [pattern] [file]

Case-insensitive search

grep -n [pattern] [file]

Show matching lines with their line numbers

grep -v [pattern] [file]

Show lines that do NOT match the pattern (invert match)

grep -E "[regex]" [file]

Use extended regular expressions

sed 's/old/new/g' [file]

Replace all occurrences of "old" with "new" in output

sed -i 's/old/new/g' [file]

Replace text in-place inside the file

awk '{print $1}' [file]

Print the first whitespace-delimited field of each line

awk -F',' '{print $2}' [file]

Print the second field using comma as delimiter (CSV)

sort [file]

Sort lines alphabetically

sort -n [file]

Sort lines numerically

sort -u [file]

Sort and remove duplicate lines

uniq [file]

Remove consecutive duplicate lines (sort first)

cut -d',' -f1 [file]

Extract the first comma-separated field from each line

tr 'a-z' 'A-Z'

Translate lowercase letters to uppercase (reads from stdin)

xargs [command]

Build and execute a command using stdin as arguments

diff [file1] [file2]

Show line-by-line differences between two files

I/O Redirection & Pipes

cmd1 | cmd2

Pipe: send stdout of cmd1 as stdin to cmd2

command > file

Redirect stdout to a file (overwrites existing content)

command >> file

Append stdout to a file

command 2> file

Redirect stderr to a file

command 2>&1

Redirect stderr to the same destination as stdout

command < file

Redirect a file as stdin to a command

command > /dev/null

Discard stdout output (send to null device)

tee [file]

Write stdin to both stdout and a file simultaneously

System Information

uname -a

Show all system information (kernel, hostname, OS, architecture)

hostname

Display or set the system's hostname

whoami

Print the current logged-in username

id

Show current user ID, group ID, and group memberships

uptime

Show how long the system has been running and load averages

df -h

Show disk space usage for all mounted filesystems (human-readable)

du -sh [directory]

Show the total disk usage of a directory

du -sh * | sort -h

List sizes of items in current directory, sorted by size

free -h

Show RAM and swap memory usage (human-readable)

date

Display the current date and time

env

List all environment variables

echo $VAR

Print the value of an environment variable

Process Management

top

Real-time interactive process viewer (q to quit)

htop

Enhanced interactive process viewer (if installed)

ps aux

List all running processes with details (CPU, memory, user)

ps aux | grep [name]

Find a specific process by name

kill [PID]

Send SIGTERM to a process, asking it to terminate gracefully

kill -9 [PID]

Force-kill a process immediately (SIGKILL)

killall [name]

Kill all processes matching a name

command &

Run a command in the background

jobs

List all background jobs in the current shell

bg [%job]

Resume a suspended job in the background

fg [%job]

Bring a background job to the foreground

nohup command &

Run a command immune to hangups (persists after logout)

Ctrl + C

Interrupt (terminate) the currently running foreground process

Ctrl + Z

Suspend the currently running foreground process

File Permissions

chmod [mode] [file]

Change file permissions using octal or symbolic mode

chmod 755 [file]

Set rwxr-xr-x (owner: all; group/others: read+execute)

chmod 644 [file]

Set rw-r--r-- (owner: read+write; group/others: read only)

chmod +x [file]

Add execute permission for all users

chmod -R 755 [directory]

Recursively change permissions for all files in a directory

chown [user]:[group] [file]

Change the owner and group of a file

chown -R [user]:[group] [dir]

Recursively change ownership of a directory

umask

Show or set the default permission mask for new files

Networking

ping [host]

Send ICMP packets to test network connectivity to a host

ping -c 4 [host]

Send exactly 4 ping packets and stop

curl [url]

Transfer data from or to a URL (prints to stdout by default)

curl -O [url]

Download a file, saving it with the original filename

curl -X POST -d '{}' [url]

Send a POST request with a JSON body

wget [url]

Download a file from the internet

ssh [user]@[host]

Open an SSH session to a remote host

ssh -p [port] [user]@[host]

SSH on a non-standard port

scp [file] [user]@[host]:[path]

Securely copy a file to a remote host

scp -r [dir] [user]@[host]:[path]

Recursively copy a directory to a remote host

ifconfig

Show network interface configuration (legacy)

ip addr

Show network interfaces and IP addresses (modern)

netstat -tulpn

Show listening TCP/UDP ports with PID info

ss -tulpn

Show socket statistics (faster modern alternative to netstat)

nslookup [domain]

Query DNS to resolve a domain name

dig [domain]

Detailed DNS lookup with query/response info

Compression & Archiving

tar -czf archive.tar.gz [dir]

Create a gzip-compressed tar archive from a directory

tar -xzf archive.tar.gz

Extract a gzip-compressed tar archive

tar -xzf archive.tar.gz -C [dir]

Extract tar archive into a specific directory

tar -tzf archive.tar.gz

List contents of a tar archive without extracting

zip -r archive.zip [dir]

Create a zip archive from a directory

unzip archive.zip

Extract a zip archive

unzip -l archive.zip

List contents of a zip archive without extracting

gzip [file]

Compress a file (replaces original with .gz file)

gunzip [file.gz]

Decompress a .gz file

Shell & Environment

export VAR=value

Set an environment variable for the current session

echo $VAR

Print the value of an environment variable

unset VAR

Remove an environment variable

alias ll='ls -la'

Create a shortcut alias for a command

source ~/.bashrc

Reload your shell configuration file without restarting

history

Show the command history for the current shell

history | grep [term]

Search your command history for a specific term

which [command]

Show the full path to a command's executable

man [command]

Open the manual (help) page for a command

sudo [command]

Run a command with superuser (root) privileges

!! sudo !!

Repeat the last command; prepend sudo to re-run it as root

Ctrl + R

Reverse-search through command history interactively

Ctrl + L

Clear the terminal screen

Tab

Auto-complete commands, file names, and paths

Package Management

apt update

Update the local package list (Debian/Ubuntu)

apt install [package]

Install a package (Debian/Ubuntu)

apt remove [package]

Remove a package (Debian/Ubuntu)

apt upgrade

Upgrade all installed packages (Debian/Ubuntu)

yum install [package]

Install a package (RHEL/CentOS/Fedora)

dnf install [package]

Install a package using DNF (modern Fedora/RHEL)

brew install [package]

Install a package using Homebrew (macOS)

brew update && brew upgrade

Update Homebrew and upgrade all installed packages (macOS)
