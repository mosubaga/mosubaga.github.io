Git Commands

The complete Git workflow — from first init to advanced collaboration — in one place.

Setup & Configuration

git config --global user.name "[name]"

Set the name attached to all your commits globally

git config --global user.email "[email]"

Set the email attached to all your commits globally

git config --global color.ui auto

Enable helpful colorized output in the terminal

git config --global core.editor [editor]

Set the default text editor for commit messages (e.g., vim, nano, code)

git config --list

Show all current Git configuration settings

git config --global alias.st status

Create a shorthand alias (e.g., git st = git status)

Create Repositories

git init

Initialize a new Git repository in the current directory

git init [project-name]

Create a new directory and initialize a Git repo inside it

git clone [url]

Clone a remote repository to a new local directory

git clone [url] [directory]

Clone a remote repository into a specific local directory

git clone --depth 1 [url]

Shallow clone — download only the most recent commit (faster)

Stage & Snapshot

git status

Show the working tree status — staged, unstaged, and untracked files

git add [file]

Stage a specific file for the next commit

git add .

Stage all changes (new, modified, deleted) in the current directory

git add -p

Interactively stage chunks of changes (patch mode)

git diff

Show unstaged changes — what's in the working tree vs the index

git diff --staged

Show staged changes — what's in the index vs the last commit

git reset [file]

Unstage a file while keeping its changes in the working tree

git restore --staged [file]

Unstage a file (modern equivalent of git reset [file])

git restore [file]

Discard changes in the working tree for a file

git commit -m "[message]"

Commit staged changes with a descriptive message

git commit -am "[message]"

Stage all tracked changes and commit in one step

git commit --amend

Modify the most recent commit (message or staged content)

git commit --amend --no-edit

Add staged changes to the last commit without changing its message

Branching

git branch

List all local branches (* marks the current branch)

git branch -a

List all local and remote-tracking branches

git branch [name]

Create a new branch at the current commit (does not switch)

git branch -d [name]

Delete a branch (only if it has been merged)

git branch -D [name]

Force-delete a branch regardless of merge status

git branch -m [old] [new]

Rename a branch

git checkout [branch]

Switch to an existing branch

git checkout -b [name]

Create a new branch and switch to it immediately

git switch [branch]

Switch to a branch (modern alternative to git checkout)

git switch -c [name]

Create and switch to a new branch (modern syntax)

git merge [branch]

Merge the specified branch into the current branch

git merge --no-ff [branch]

Merge without fast-forwarding (preserves branch history)

git rebase [branch]

Reapply commits from current branch on top of another branch

git rebase -i HEAD~[n]

Interactively rebase the last n commits (squash, reorder, edit)

git cherry-pick [commit]

Apply the changes from a specific commit onto the current branch

Stash

git stash

Stash current uncommitted changes and restore a clean working tree

git stash push -m "[message]"

Stash changes with a descriptive message

git stash list

List all saved stashes

git stash pop

Apply the most recent stash and remove it from the stash list

git stash apply stash@{n}

Apply a specific stash without removing it from the list

git stash drop stash@{n}

Delete a specific stash entry

git stash clear

Remove all stash entries

git stash show -p stash@{n}

Show the diff of a specific stash entry

Inspect & Review History

git log

Show the full commit history with author, date, and message

git log --oneline

Show a compact one-line-per-commit history

git log --oneline --graph --all

Show a visual branch graph of the full history

git log --follow [file]

Show commit history for a specific file, following renames

git log -n 10

Show only the last 10 commits

git log --author="[name]"

Show commits by a specific author

git log --since="2 weeks ago"

Show commits from the last two weeks

git diff [branch1]...[branch2]

Show changes between two branches

git show [commit]

Show the changes introduced by a specific commit

git blame [file]

Show who last modified each line of a file and when

git shortlog -sn

Show commit count per author, sorted by number of commits

git reflog

Show a log of all HEAD movements — useful for recovering lost commits

Undo Changes

git reset HEAD~1

Undo the last commit, keeping all changes unstaged

git reset --soft HEAD~1

Undo the last commit, keeping all changes staged

git reset --hard HEAD~1

Undo the last commit and discard all changes (destructive)

git revert [commit]

Create a new commit that undoes changes from a specific commit (safe for shared history)

git restore [file]

Discard changes to a file in the working directory

git restore --staged [file]

Unstage a file (keep the working tree changes)

git clean -fd

Remove untracked files and directories from the working tree

git clean -n

Dry-run — show which files would be removed by git clean

Ignore Files

.gitignore

File listing patterns of files/dirs Git should not track

git ls-files --others --ignored --exclude-standard

List all ignored files in the repository

git check-ignore -v [file]

Show which .gitignore rule is causing a file to be ignored

git rm --cached [file]

Stop tracking a file without deleting it from disk

Tags

git tag

List all tags in the repository

git tag [name]

Create a lightweight tag at the current commit

git tag -a [name] -m "[message]"

Create an annotated tag with a message (recommended for releases)

git tag -a [name] [commit]

Tag a specific past commit

git push origin [tag]

Push a specific tag to the remote repository

git push origin --tags

Push all local tags to the remote repository

git tag -d [name]

Delete a local tag

git push origin --delete [tag]

Delete a tag from the remote repository

Synchronize with Remote

git remote -v

List all remote connections with their fetch/push URLs

git remote add [name] [url]

Add a new remote connection named "name"

git remote remove [name]

Remove a remote connection

git remote set-url origin [url]

Change the URL of the origin remote

git fetch [remote]

Download changes from remote without merging into local branch

git fetch --all

Fetch from all configured remote repositories

git pull

Fetch from the tracking remote and merge into the current branch

git pull --rebase

Fetch and rebase local commits on top of the remote (cleaner history)

git push [remote] [branch]

Push local branch to the specified remote

git push -u origin [branch]

Push and set the upstream tracking branch (needed once per new branch)

git push --force-with-lease

Force push safely — aborts if remote has changes you haven't fetched

git push origin --delete [branch]

Delete a branch on the remote repository
