Quick Glossary

Search the terms that show up most often in Obsidian tutorials.

Vault

A folder of Markdown notes and attachments that Obsidian opens as one workspace.

Markdown

Plain text formatting for headings, lists, links, code, quotes, and more.

Wikilink

The [[Note name]] link style Obsidian uses by default for internal links.

Backlinks

A list of notes that link to the current note.

Graph view

A visual map where notes are nodes and internal links are lines.

Canvas

A visual workspace for arranging notes, media, web pages, and connections in 2D space.

Properties

Structured metadata stored at the top of notes, such as tags, status, date, or source.

Daily notes

Notes automatically named by date, often used for capture, logs, and planning.

Community plugins

Optional extensions made by the Obsidian community. Install only what you trust and need.
