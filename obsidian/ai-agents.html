AI-assisted notes

Use AI agents to improve your vault, while keeping the vault yours.

Because an Obsidian vault is just files, tools like Codex or Claude Code can read and edit notes when you point them at the vault folder. The trick is to give them narrow tasks and review their changes.

Good AI Agent Tasks

AI is strongest when the source material is already in your vault and the output format is clear.

Summarize and link

Ask an agent to summarize a folder of lecture notes and suggest links between repeated ideas.

Generate study questions

Turn notes into practice questions, flashcard prompts, or problem sets with answers hidden below.

Clean structure

Rename notes, add properties, build index pages, and find orphan notes that need links.

Build project docs

Convert meeting notes into status updates, decision logs, or business plan sections.

Extract action items

Scan meeting notes for owners, due dates, and unresolved questions.

Write from your notes

Draft essays, reports, documentation, or outlines using only vault notes as source material.

A Safe Agent Workflow

Back up first

Use Git, Time Machine, Obsidian Sync version history, or another backup before large edits.

Give a small scope

Point the agent to one folder or a short list of notes, not your entire life archive.

Ask for a diff or summary

Before accepting edits, review changed files and check that citations or links are real.

Keep human judgment

Let AI suggest structure and questions. You decide what is true, important, and worth keeping.

Privacy check

Local files can still be sent to an AI service if your tool uses a cloud model. Do not include private, confidential, student, patient, client, or company-sensitive information unless your tool and policy allow it.

Prompt Patterns

These prompts work well because they tell the agent where to look, what to produce, and what to avoid.

Link suggester

Review the notes in 30 Resources/Physics.
Suggest internal links between related notes.
Do not rewrite notes yet.
Return a list with:
- source note
- suggested link target
- reason for the link

create 20 active-recall questions.
