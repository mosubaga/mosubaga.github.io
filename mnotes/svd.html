SVD in Pastel Geometry

Break a Matrix into Rotations and Stretching

Singular Value Decomposition (SVD) rewrites any matrix as a rotation, a scaling, and another rotation. It reveals the most important directions in your data and how much each one matters.

1) The Definition

For any matrix $A \in \mathbb{R}^{m \times n}$, SVD factorizes it into three pieces: an orthogonal matrix $U$, a diagonal matrix $\Sigma$, and another orthogonal matrix $V^T$.

$$A = U \Sigma V^T$$
$U$ (left singular vectors) Orthonormal directions in the output space.
$\Sigma$ (singular values) Nonnegative scales that say how much each direction stretches.
$V$ (right singular vectors) Orthonormal directions in the input space.
Rank insight The number of nonzero singular values equals the rank of $A$.

2) Geometry and Intuition

Think of $A$ as a transformation of a unit sphere. The SVD tells you that $A$ turns the sphere into an ellipsoid by rotating, stretching by the singular values, and rotating again.

Rotation 1

$V^T$ lines up the input with the principal axes.

Scaling

$\Sigma$ stretches or compresses along each axis.

Rotation 2

$U$ rotates into the final output orientation.

3) Why SVD Matters in Machine Learning

SVD powers dimensionality reduction, noise filtering, and fast approximations. Keeping the top $k$ singular values produces the best low-rank approximation to $A$.

$$A_k = U_k \Sigma_k V_k^T$$

Principal Component Analysis uses SVD to find the most informative directions.
