Measure Theory Notes

Introduction

Measure theory is a branch of mathematics primarily concerned with measuring the size of subsets of a given set. While classical geometry provides intuitive ways to calculate the length, area, or volume of ordinary bodies like cubes or balls, measure theory provides a rigorous grounding for these concepts when dealing with much more complex or pathological sets.

Here is a brief introduction to the subject:

1. The Motivation: Why Measure Theory?

In the early 20th century, mathematicians realized that classical methods of assigning volume, such as Jordan measure, were insufficient for sets that arise as limits or through countable operations. Certain pathological subsets also exist, such as those involved in the Banach-Tarski paradox or non-measurable sets constructed using the axiom of choice, to which a sensible volume cannot be assigned. Measure theory provides a framework to exclude these pathological sets and rigorously define integrals over nicer ones.

2. Core Components of a Measure Space

To define a measure rigorously, one needs three components, collectively known as a measure space \((X, \mathcal{F}, \mu)\):

  • The Set (\(X\)): The space of all possible points or states.
  • The \(\sigma\)-algebra (\(\mathcal{F}\)): Also called a \(\sigma\)-field, this is a collection of nice enough subsets of \(X\) that we are allowed to measure. To be a \(\sigma\)-algebra, the collection must be closed under complementation and countable unions.
  • The Measure (\(\mu\)): A function that maps sets in the \(\sigma\)-algebra to non-negative numbers \([0, \infty]\). A valid measure must satisfy countable additivity, meaning the measure of a union of disjoint sets is equal to the sum of their individual measures.

3. Important Examples of Measures

  • Lebesgue Measure: This is the most natural measure for Euclidean space \((\mathbb{R}^n)\), returning the standard volume of a subset. It extends the Riemann-Darboux theory used in calculus to a much broader class of sets.
  • Counting Measure: Used when a set is countable, this measure simply counts the number of points in a subset.
  • Probability Measure: A special case where the total measure of the entire space is exactly \(1\). This allows measure theory to provide a rigorous foundation for probability, where sets are interpreted as events and measures as probabilities.

4. Integration and Limits

One of the primary benefits of measure theory is that it allows for a more powerful theory of integration. The Lebesgue integral is more robust than the Riemann integral because it behaves much better with respect to limits.

In the Lebesgue theory, features are often preserved under limits, as exemplified by two fundamental theorems:

  • Monotone Convergence Theorem: Allows the interchange of limits and integrals for increasing sequences of non-negative functions.
  • Dominated Convergence Theorem: Allows the interchange of limits and integrals when the sequence is dominated by another integrable function.

5. Littlewood's Three Principles

To help students understand the Lebesgue philosophy, the mathematician J.E. Littlewood proposed three informal heuristics:

  1. Every measurable set is nearly a finite sum of intervals.
  2. Every absolutely integrable function is nearly continuous.
  3. Every pointwise convergent sequence of functions is nearly uniformly convergent.

These principles highlight that while measure theory deals with very general and potentially messy objects, they can usually be approximated by much simpler, classical objects.

From Lebesgue Measure to the Lebesgue Integral

The transition from Lebesgue measure to the Lebesgue integral is a systematic process that uses the size of sets as a weighting mechanism to define the area under a curve for a much broader class of functions than the classical Riemann integral.

The construction follows these key stages:

1. The Building Block: Indicator Functions

The process begins with the indicator function \(1_A\), also called a characteristic function, which takes the value \(1\) if a point \(x\) is in set \(A\) and \(0\) otherwise. The integral of an indicator function is defined simply as the measure of the set itself:

\[ \int 1_A \, d\mu = \mu(A) \]

This establishes that the area of a set with height \(1\) is exactly its measure, whether that means length, area, or volume.

2. Simple Functions

A simple function is a finite linear combination of indicator functions: \(f = \sum_{i=1}^n c_i 1_{A_i}\), where each \(A_i\) is a measurable set. Based on the requirement that integration should be linear, the integral of a simple function is defined as the weighted sum of the measures of those sets:

\[ \int f \, d\mu = \sum_{i=1}^n c_i \mu(A_i) \]

3. Unsigned Measurable Functions

To integrate more complex non-negative functions, the theory uses simple functions to approximate them from below. The integral of an unsigned measurable function \(f\) is defined as the supremum of the integrals of all simple functions \(\phi\) that are bounded by \(f\):

\[ \int f \, d\mu = \sup \left\{ \int \phi \, d\mu : 0 \le \phi \le f, \phi \text{ is simple} \right\} \]

This ensures that as the simple functions become more accurate approximations of \(f\), their integrals converge to the true area under the graph.

4. General Integrable Functions

Finally, any real-valued function is integrated by splitting it into its positive and negative parts, where \(f = f^+ - f^-\). A function is considered Lebesgue integrable if the integrals of both parts are finite:

\[ \int f \, d\mu = \int f^+ \, d\mu - \int f^- \, d\mu \]

Key Shift: Range vs. Domain

The fundamental difference between the Riemann and Lebesgue integrals lies in how they partition the function:

  • Riemann Integration: Partitions the domain, the x-axis, into small intervals. This fails if the function is too discontinuous to be handled by intervals.
  • Lebesgue Integration: Partitions the range, the y-axis, into small intervals. It then uses the Lebesgue measure to calculate the size of the inverse image, the set of points in the domain that map to a specific value in the range. Because Lebesgue measure can measure extremely complex sets, not just intervals, it can integrate pathological functions like the Dirichlet function that the Riemann integral cannot.

The Payoff: Convergence Theorems

Because the Lebesgue integral is built directly on a countably additive measure, it behaves much better than the Riemann integral with respect to limits. Fundamental results such as the Monotone Convergence Theorem and the Dominated Convergence Theorem allow mathematicians to interchange limits and integrals under broad conditions, which is essential for modern analysis and probability theory.

Example: Integrating a Parabola

The Lebesgue integral of a simple parabola, such as \(f(x) = x^2\), can be shown explicitly, and it yields the same result as the Riemann integral.

For well-behaved functions, specifically those that are continuous on a closed, bounded interval, the Lebesgue theory acts as a completion of the Riemann theory. In these cases, the two methods are compatible, and their values coincide.

The Specific Example: \(f(x) = x^2\) on \([0,1]\)

The sources provide a detailed comparison using the parabola on the interval \([0,1]\):

  • The Riemann Approach: To compute the Riemann integral, partition the domain, the x-axis, into subintervals. Using a partition of \(n\) equal parts, the upper and lower sums are calculated based on the heights of rectangles at the endpoints of these intervals. As \(n\) approaches infinity, both the lower and upper Riemann sums converge to \(1/3\).
  • The Lebesgue Approach: The Lebesgue integral is defined by approximating the function from below using simple functions, piecewise constant functions defined on measurable sets. For the parabola, the Lebesgue integral is calculated as:

\[ \int_{0}^{1} x^2 \, dm = \frac{1}{3} \]

This matches the Riemann result because \(f(x) = x^2\) is continuous, and every continuous function on a closed bounded interval is both Riemann and Lebesgue integrable.

Conceptual Difference: Range vs. Domain

While the result is the same, the philosophy of the calculation differs:

  • Riemann partitions the domain.
  • Lebesgue partitions the range.

It then uses the measure of the sets in the domain that map into those range intervals to calculate the area.

Henri Lebesgue famously illustrated this difference using a money analogy: a Riemann integral is like counting a pile of coins by picking them up one by one as you find them, while a Lebesgue integral is like sorting the coins into heaps of the same value first and then multiplying the value of each heap by the number of coins in it.

For a smooth curve like a parabola, both counting methods are equally effective, but the Lebesgue method remains valid even for much more pathological or discontinuous functions where the Riemann method fails.
