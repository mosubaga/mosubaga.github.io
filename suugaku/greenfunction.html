Differential Equations

Green's Function

A Green's function is the response of a linear operator to a point source. Once it is known, the solution to the same operator driven by any source is an integral over the source, weighted by the Green's function.

Overview

Let \(\mathcal{L}\) be a linear differential operator acting on functions of \(x\) on a domain \(\Omega\) with prescribed boundary conditions. The Green's function \(G(x, x')\) satisfies

\[ \mathcal{L}_x \, G(x, x') = \delta(x - x'), \]

subject to the same homogeneous boundary conditions. Here \(\delta(x-x')\) is the Dirac delta — a unit impulse concentrated at the source point \(x'\).

The key result is that the solution \(u(x)\) to \(\mathcal{L} u = f(x)\) is then obtained by superposition:

\[ u(x) = \int_\Omega G(x, x')\, f(x')\, dx'. \]

Each colored curve is the Green's function \(G(x, x')\) for a different source location \(x'\). The white curve is their superposition \(u(x) = \int G(x,x') f(x')\, dx'\) for a smooth source \(f\).

1. Why Point Sources?

The Dirac delta \(\delta(x - x')\) is the identity element of convolution:

\[ \int \delta(x - x') f(x')\, dx' = f(x). \]

Because \(\mathcal{L}\) is linear, we can think of any source \(f(x)\) as a continuous superposition of point sources. The solution to each point-source problem is \(G(x, x')\). Linearity then guarantees that the solution to the original problem is the same superposition of Green's functions:

\[ \mathcal{L}\, u = \mathcal{L} \int G(x,x') f(x')\, dx' = \int \underbrace{\mathcal{L}_x G(x,x')}_{\delta(x-x')} f(x')\, dx' = f(x). \]

The method therefore converts an ODE (or PDE) boundary-value problem into an explicit integral — no shooting method or iteration required.

2. Worked Example — The Simple Boundary-Value Problem

We solve the ODE

\[ -u''(x) = f(x), \qquad x \in (0,1), \qquad u(0) = u(1) = 0. \]

The Green's function \(G(x,x')\) for the operator \(\mathcal{L} = -d^2/dx^2\) on \([0,1]\) with Dirichlet conditions satisfies

\[ -G_{xx}(x, x') = \delta(x - x'), \qquad G(0, x') = G(1, x') = 0. \]

3. Deriving \(G(x, x')\) Step by Step

Step 1 — Solve in each subinterval. Away from the source point \(x'\), the equation becomes \(-G_{xx} = 0\), so \(G\) is piecewise linear:

\[ G(x, x') = \begin{cases} A\, x & 0 \le x < x', \\ B\,(1 - x) & x' < x \le 1. \end{cases} \]

The boundary conditions \(G(0,x')=0\) and \(G(1,x')=0\) are already satisfied.

Step 2 — Continuity at \(x'\). \(G\) must be continuous:

\[ A\, x' = B\,(1 - x'). \]

Step 3 — Jump condition on the derivative. Integrating \(-G_{xx} = \delta(x-x')\) over a tiny interval around \(x'\) gives a unit jump in the derivative:

\[ G_x(x'^+, x') - G_x(x'^-, x') = -1, \qquad \text{i.e.}\quad -B - A = -1, \quad\text{so}\quad A + B = 1. \]

Step 4 — Solve for \(A\) and \(B\). From \(Ax' = B(1-x')\) and \(A + B = 1\):

\[ B = \frac{x'}{1} = x', \qquad A = 1 - x'. \]

Result.

\[ G(x, x') = \begin{cases} x\,(1-x') & 0 \le x \le x', \\ x'\,(1-x) & x' \le x \le 1. \end{cases} \]

Notice that \(G(x,x') = G(x',x)\) — the Green's function is symmetric (the operator is self-adjoint). The kink at \(x = x'\) is the signature of the delta-function source.

4. Applying the Formula — A Specific Source

Let \(f(x) = 1\) (uniform distributed load). The solution is

\[ u(x) = \int_0^1 G(x,x')\cdot 1 \; dx' = \int_0^x x'(1-x)\,dx' + \int_x^1 x(1-x')\,dx'. \]

Evaluating each integral:

Left piece

\[ (1-x)\int_0^x x'\,dx' = (1-x)\frac{x^2}{2} \]

Right piece

\[ x\int_x^1 (1-x')\,dx' = x\,\frac{(1-x)^2}{2} \]

Adding both:

\[ u(x) = \frac{x(1-x)}{2}\bigl(x + (1-x)\bigr) = \frac{x(1-x)}{2}. \]

We can verify: \(-u''(x) = -(- 1) = 1 = f(x)\) ✓, and \(u(0)=u(1)=0\) ✓. The parabola \(x(1-x)/2\) is the familiar deflection of a simply supported beam under a uniform load.

Symmetry and Self-Adjointness

For a self-adjoint operator (one that satisfies \(\langle u, \mathcal{L}v \rangle = \langle \mathcal{L}u, v \rangle\)), the Green's function is symmetric:

\[ G(x, x') = G(x', x). \]

Physically this is reciprocity: the displacement at \(x\) due to a unit force at \(x'\) equals the displacement at \(x'\) due to a unit force at \(x\). This is Betti's reciprocal theorem in structural mechanics.

Spectral representation. For a self-adjoint operator with eigenfunctions \(\phi_n\) satisfying \(\mathcal{L}\phi_n = \lambda_n \phi_n\), the Green's function can be expanded as \[ G(x, x') = \sum_n \frac{\phi_n(x)\,\phi_n(x')}{\lambda_n}. \] Each eigenmode contributes independently; the Green's function inverts the operator by dividing by each eigenvalue. This is the spectral analogue of \(A^{-1} = \sum_n \mathbf{v}_n \mathbf{v}_n^\top / \lambda_n\) for matrices.

5. Green's Functions in Higher Dimensions and Physics

The concept generalizes naturally to PDEs. For the Laplacian in \(\mathbb{R}^3\), the free-space Green's function satisfies

\[ -\nabla^2 G(\mathbf{r}, \mathbf{r}') = \delta^{(3)}(\mathbf{r} - \mathbf{r}'), \qquad G(\mathbf{r}, \mathbf{r}') = \frac{1}{4\pi|\mathbf{r}-\mathbf{r}'|}. \]

This is the electrostatic potential at \(\mathbf{r}\) due to a unit point charge at \(\mathbf{r}'\). The solution to Poisson's equation \(-\nabla^2 \phi = \rho/\varepsilon_0\) is then

\[ \phi(\mathbf{r}) = \frac{1}{4\pi\varepsilon_0} \int \frac{\rho(\mathbf{r}')}{|\mathbf{r}-\mathbf{r}'|}\, d^3r', \]

exactly Coulomb's law for a charge distribution. The same construction applies to the heat equation, the wave equation, and the Schrödinger equation.

6. Real-World Applications

  • Electrostatics and Magnetostatics. Computing the electric field or magnetic vector potential due to an arbitrary charge or current distribution via Coulomb/Biot-Savart integrals.
  • Structural Mechanics. The deflection of a beam or plate under a distributed load is the Green's function convolution. Finite element methods approximate the Green's function on complex geometries.
  • Quantum Mechanics. The retarded Green's function (propagator) describes how a quantum state evolves. The Born series for scattering amplitudes is a perturbative expansion of the full propagator.
  • Seismology. The seismic response of the Earth to an earthquake is modeled using the Green's function of the elastic wave equation. Observed waveforms are deconvolved with the Green's function to infer source properties.
  • Heat Conduction. Temperature at a point due to a heat source or initial condition. The heat kernel \(G(x,t;x',t') = e^{-(x-x')^2/4(t-t')} / \sqrt{4\pi(t-t')}\) is the Green's function of the diffusion equation.
  • Image Processing. Gaussian blur is convolution with the heat-equation Green's function. Edge detection uses Green's functions of differential operators in disguise.
