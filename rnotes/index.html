What is Ruby?

Ruby is an elegant, high-level programming language designed with developer happiness in mind. Created by Yukihiro Matsumoto (Matz) in the mid-1990s, Ruby combines the best features of Perl, Smalltalk, Eiffel, Ada, and Lisp to create a language that is natural to read and write.

# Ruby's elegant syntax makes code readable
5.times { print "We love Ruby! " }

# Everything is an object
"hello".upcase.reverse # => "OLLEH"

# Blocks make iteration beautiful
[1, 2, 3, 4, 5].map { |n| n * 2 }

Ruby emphasizes human-friendly syntax over machine efficiency, making it one of the most enjoyable languages to write. Its philosophy centers on the principle that programming should be fun and that software development is fundamentally about people, not machines.

Installing Ruby

You will need to install Ruby from Ruby.org. Follow the instructions depending on your OS.

Once you install Ruby, check if it was properly installed by:

ruby -v
