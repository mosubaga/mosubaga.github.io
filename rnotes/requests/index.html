GET, POST, PUT, and all HTTP methods in Ruby

Ruby should have net/http gem installed by default. All you need is this gem to send HTTP requests.

Example of a GET request


require 'net/http'
require 'uri'
require 'json'

uri = URI('https://www.api-example.com/api/v1/id')
http = Net::HTTP.new(uri.host, uri.port)
http.use_ssl = true

# GET request
response = http.get(uri.path)

puts "GET Response: #{response.body}"

Example of a POST request


require 'net/http'
require 'uri'
require 'json'

uri = URI('https://www.api-example.com/api/v1')
http = Net::HTTP.new(uri.host, uri.port)
http.use_ssl = true

# Create POST request
request = Net::HTTP::Post.new(uri.path)
request['Content-Type'] = 'application/json'

# Set request body
request.body = {
  title: 'My Post',
  body: 'This is the post content',
  userId: 1
}.to_json

# Send request
response = http.request(request)

puts "Status: #{response.code}"
puts "Response: #{response.body}"

Example of a PUT request


require 'net/http'
require 'uri'
require 'json'

uri = URI('https://jsonplaceholder.typicode.com/posts')
http = Net::HTTP.new(uri.host, uri.port)
http.use_ssl = true

# PUT request
request = Net::HTTP::Put.new(uri.path)
request['Content-Type'] = 'application/json'

request.body = {
  title: 'Updated Title'
}.to_json

response = http.request(request)

puts "PUT Response: #{response.body}"

Example of a DELETE request


require 'net/http'
require 'uri'
uri = URI('https://jsonplaceholder.typicode.com/posts/id/1')
http = Net::HTTP.new(uri.host, uri.port)
http.use_ssl = true

request = Net::HTTP::Delete.new(uri.path)
response = http.request(request)

puts "Deleted: #{response.code}"

Using HTTParty

HTTParty makes HTTP requests simple and fun with its clean, intuitive API. It's the most popular choice for Ruby developers. Refer to HTTParty for more information.

gem install httparty
