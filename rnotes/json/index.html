Parsing JSON

JSON support is included in Ruby's standard libraryâ€”just require it and start coding! Ruby's JSON parser is implemented in C for maximum performance.

Here's how you can parse a JSON string:

require 'json'

# JSON string
json_string = '{"name":"John Doe","age":30,"email":"john@example.com"}'

# Parse JSON string into Ruby hash
data = JSON.parse(json_string)

# Access data like a regular hash
puts "Name: #{data['name']}"
puts "Age: #{data['age']}"
puts "Email: #{data['email']}"

# Output:
# Name: John Doe
# Age: 30
# Email: john@example.com

Here's how you can parse a JSON file:

require 'json'

# Read JSON file
file_content = File.read('data.json')

# Parse the JSON data = JSON.parse(file_content)
puts data

# Alternative: One-liner
data = JSON.parse(File.read('data.json'))
