Control Structures

Ruby provides standard control structures like `if`, `else`, `elsif`, and `unless`. 

# if-else statement
age = 18
if age >= 18
  puts "You are an adult."
else
  puts "You are a minor."
end

# unless (opposite of if)
unless age < 18
  puts "You can enter."
end

Loops

Ruby has many ways to iterate. Common loops include `while`, `until`, and various iterators like `each` and `times`. 

# while loop
i = 0
while i < 5
  puts i
  i += 1
end

# each iterator (the Ruby way)
[1, 2, 3].each do |num|
  puts num
end

# times iterator
5.times do |i|
  puts "Iteration #{i}"
end

File Read/Write

Ruby makes it easy to read from and write to files using the `File` class. 

# Writing to a file
File.open("example.txt", "w") do |file|
  file.puts "Hello, Ruby!"
end

# Reading from a file
content = File.read("example.txt")
puts content

# Reading line by line
File.foreach("example.txt") do |line|
  puts line
end
