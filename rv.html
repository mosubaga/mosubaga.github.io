Discrete Random Variables

1. Definition of a Random Variable

A random variable is defined as a real-valued function defined on the sample space of an experiment. Rather than looking at every detail of an outcome, we use random variables to focus on a specific numerical quantity of interest (for example, the sum of two dice or the number of heads in coin flips).

2. Discrete Random Variables and PMFs

A random variable is considered discrete if it can take on at most a countable number of possible values.

  • Probability Mass Function (PMF): For a discrete random variable \(X\), the PMF \(p(x)\) is defined as the probability that \(X\) takes on the value \(x\): \(p(x) = P(X = x)\).
  • The sum of these probabilities across all possible values must always equal 1.

3. Summarizing Random Variables

The chapter introduces two primary ways to summarize the behavior of a distribution:

  • Expected Value (Mean): Denoted as \(E[X]\) or \(\mu\), the expected value is the probability-weighted average of all possible values. It represents the "center of gravity" of the distribution.
  • Variance and Standard Deviation: The variance (\(Var(X)\)) measures the "spread" or dispersion of the values around the mean. The standard deviation is simply the square root of the variance.

4. Key Discrete Probability Distributions

A significant portion of the chapter is dedicated to several "named" distributions that occur frequently in real-world applications:

  • Bernoulli: A trial with only two possible outcomes, success or failure.
  • Binomial: Represents the number of successes in \(n\) independent Bernoulli trials.
  • Poisson: Often used to model the number of events occurring in a fixed interval of time or space (e.g., typographical errors on a page).
  • Geometric and Negative Binomial: The Geometric distribution models the number of trials until the first success, while the Negative Binomial models the trials until the \(r\)th success.
  • Hypergeometric: Used when sampling from a finite population without replacement.

5. Cumulative Distribution Function (CDF)

The Cumulative Distribution Function (\(F(x)\)) is defined for any value \(x\) as the probability that the random variable is less than or equal to \(x\): \(F(x) = P(X \leq x)\).

Key properties of a discrete CDF include:

  1. It is a nondecreasing function.
  2. It ranges from 0 to 1.
  3. It is right-continuous and typically resembles a "staircase" for discrete variables.

Continuous Random Variables

1. Introduction and Probability Density Functions (PDF)

A random variable \(X\) is defined as continuous (or absolutely continuous) if there exists a non-negative function \(f(x)\), known as the probability density function, such that the probability of \(X\) falling within any set \(B\) is the integral of \(f(x)\) over that set.

  • Key Property: The probability that a continuous random variable assumes any one fixed, specific value is zero. Consequently, for any interval, it does not matter whether the endpoints are included in the probability calculation (e.g., \(P(a \leq X \leq b) = P(a < X < b)\)).
  • Cumulative Distribution Function (CDF): For a continuous variable, the CDF \(F(x)\) is the integral of the PDF from \(-\infty\) to \(x\). It is a differentiable function, and its derivative is the PDF (\(F'(x) = f(x)\)).

2. Expectation and Variance

The chapter adapts the summaries of center and spread to the continuous case:

  • Expected Value (Mean): Defined as the integral \(E[X] = \int_{-\infty}^{\infty} x f(x) dx\). It represents the weighted average of all possible values.
  • Variance and Standard Deviation: The variance is defined as \(Var(X) = E[(X - \mu)^2]\), measuring the spread of the distribution around the mean.

3. Key Continuous Distributions

A major portion of the chapter is dedicated to specific families of distributions used to model real-world phenomena:

  • The Uniform Random Variable: Models scenarios where a variable is "equally likely" to be anywhere within a specified interval \((a, b)\).
  • Normal Random Variables: Characterized by a symmetric, bell-shaped density curve. It is the most widely used model for continuous measurements. This section also covers the Normal Approximation to the Binomial Distribution, used when the number of trials \(n\) is large.
  • Exponential Random Variables: Frequently used to model lifetimes, such as the time until a battery wears out or the time between events. This section introduces Hazard Rate Functions.
  • Other Distributions: The chapter provides brief overviews of the Gamma, Weibull, Cauchy, Beta, and Pareto distributions.

4. Distribution of a Function of a Random Variable

The final section provides mathematical techniques for determining the probability distribution of a new random variable \(Y\) that is defined as a function of an existing random variable \(X\) (e.g., \(Y = X^2\) or \(Y = e^X\)).
