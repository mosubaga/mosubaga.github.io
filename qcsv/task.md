# Task

Please create a website that looks like Microsoft Excel using the boilerplate in the `qcsv` folder

# Format

1. Please use index.html as the HTML.
2. Please create an Excel like web page where user can open, edit, and save a csv file.
3. Only edit csv file, no need for the ability to work with xlsx or xls file. Only csv file.
4. The user can drag/drop a csv file or user can click on the button and browse for the csv file to open. There will be cel editor where use can edit the csv file and save the file.

# Structure
1. The page has an Excel or Mac's Number type style. The style info has to go to `css/style.css`.
2. All script (javascript) has to be stored in `js/script.js`.
3. The HTML has to refer to style and script from the files above.
