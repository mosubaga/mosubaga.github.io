Legendre Transform (Lagrangian & Hamiltonian Mechanics)

1. Big Picture

A Legendre transform is a mathematical tool that changes the independent variables of a function from one set to a conjugate set.

Key Idea

It lets us replace a variable with its conjugate variable while preserving physical meaning.

2. Conjugate Variables

Two variables are conjugate if they appear in a differential of the form

\begin{equation} df = u\,dx + w\,dy \end{equation}

Then the conjugate quantities are given by

\begin{equation} u = \frac{\partial f}{\partial x}, \qquad w = \frac{\partial f}{\partial y}. \end{equation}

3. Definition of Legendre Transform

Given a function \(f(x,y)\), define its Legendre transform by

\begin{equation} g(x,w) = f(x,y) - wy, \end{equation}

where

\begin{equation} w = \frac{\partial f}{\partial y}. \end{equation}

4. From Lagrangian to Hamiltonian

Start with a Lagrangian \(L(q,\dot{q})\). Define the momentum by

\begin{equation} p = \frac{\partial L}{\partial \dot{q}}. \end{equation}

The Legendre transform then gives the Hamiltonian

\begin{equation} H(q,p) = p\dot{q} - L(q,\dot{q}). \end{equation}

5. Example

Consider the Lagrangian

\begin{equation} L = \frac{1}{2}m\dot{q}^{2} - U(q). \end{equation}

Then

\begin{equation} p = m\dot{q}, \qquad \dot{q} = \frac{p}{m}. \end{equation}

So the Hamiltonian becomes

\begin{equation} H = \frac{p^{2}}{2m} + U(q). \end{equation}

6. Key Insight

The Legendre transform replaces velocity with momentum:

\begin{equation} \dot{q} \to p. \end{equation}

7. Summary

\begin{equation} H = p\dot{q} - L \end{equation}
\begin{equation} (q,\dot{q}) \to (q,p). \end{equation}

8. Interpretation

It changes perspective:

Lagrangian mechanics is velocity-based, while Hamiltonian mechanics is momentum-based.
