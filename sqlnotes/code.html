Getting Rows (SELECT)

The SELECT statement is used to retrieve data from database tables.

Select All Columns

-- Get all columns from users table
SELECT * FROM users;

Select Specific Columns

-- Get only name and email columns
SELECT name, email FROM users;

Select with Conditions

-- Get users where age is greater than 18
SELECT name, age FROM users WHERE age > 18;

-- Get users with specific email domain
SELECT * FROM users WHERE email LIKE '%@gmail.com';

Add Rows (INSERT)

The INSERT statement is used to add new records to a table.

Insert Single Row

-- Insert a new user
INSERT INTO users (name, email, age)
VALUES ('John Doe', 'john@example.com', 25);

Insert Multiple Rows

-- Insert multiple users at once
INSERT INTO users (name, email, age)
VALUES 
  ('Alice Smith', 'alice@example.com', 30),
  ('Bob Johnson', 'bob@example.com', 28),
  ('Carol White', 'carol@example.com', 35);

Modify Rows (UPDATE)

The UPDATE statement is used to modify existing records in a table.

Update Single Column

-- Update email for a specific user
UPDATE users
SET email = 'newemail@example.com'
WHERE name = 'John Doe';

Update Multiple Columns

-- Update multiple fields at once
UPDATE users
SET email = 'updated@example.com', age = 26
WHERE name = 'John Doe';

Update with Condition

-- Increase age by 1 for all users over 25
UPDATE users
SET age = age + 1
WHERE age > 25;

Delete Rows (DELETE)

The DELETE statement is used to remove records from a table.

Delete Specific Row

-- Delete a user by name
DELETE FROM users WHERE name = 'John Doe';

Delete with Multiple Conditions

-- Delete users younger than 18
DELETE FROM users WHERE age < 18;

-- Delete inactive users
DELETE FROM users WHERE last_login < '2023-01-01';

Delete All Rows (Use with Caution!)

-- Delete all records from the table
DELETE FROM users;

Querying Multiple Tables (JOIN)

JOIN operations allow you to combine rows from two or more tables based on related columns.

INNER JOIN

Returns records that have matching values in both tables.

-- Get users with their orders
SELECT users.name, orders.order_id, orders.total
FROM users
INNER JOIN orders ON users.user_id = orders.user_id;

LEFT JOIN

Returns all records from the left table, and matching records from the right table.

-- Get all users and their orders (if any)
SELECT users.name, orders.order_id, orders.total
FROM users
LEFT JOIN orders ON users.user_id = orders.user_id;

Multiple JOINs

Join more than two tables together.

-- Get users, their orders, and product details
SELECT 
  users.name, 
  orders.order_id, 
  products.product_name,
  products.price
FROM users
INNER JOIN orders ON users.user_id = orders.user_id
INNER JOIN products ON orders.product_id = products.product_id;

JOIN with WHERE Clause

Filter results from joined tables.

-- Get orders over $100 with user information
SELECT users.name, orders.order_id, orders.total
FROM users
INNER JOIN orders ON users.user_id = orders.user_id
WHERE orders.total > 100
ORDER BY orders.total DESC;
